A simple batch file intended to allow user to check if their account had been breached through the Have I Been Pwned service.


Function

  • Check whether your account had been breached
  • Get your account pastes

How to Use

Download .bat file and double click it.
Ensure the callurl.cmd file is on the same folder as the .bat file.


Download


Download | GitHub


Details


Produces a Report located at the place where you put the .bat file

Types of Report:

  • Truncated Report - Report only the Breach Name
  • Full Report - Report more details about the breach
