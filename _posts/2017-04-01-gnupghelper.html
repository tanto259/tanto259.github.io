A simple batch file intended to help those who have GnuPG in Windows


Function

  • Encrypt and Decrypt a File
  • Sign and Verify a File with the ability to set policy URL
  • Import Keys
  • Create Revocation Certificate and Import Them
  • Signing Keys with --ask-cert-level as default with the ability to set policy URL
  • Deleting Keys

How to Use

Download .bat file and double click it


Download


Dependencies

Built for GnuPG Modern Release

GnuPG 2.1 (tested on 2.1.19)

Gpg4win 3 Beta (untested)

Usage for GnuPG Stable 2.0 and Classic 1.4 are untested and unsupported.

Download | GitHub


Details

  • Support Windows XP Service Pack 2 and later
  • .bat file is signed with GnuPG key id 0xEBE358FC
  • Require internet connection when checking for update
  • There is an in-app changelog
Integrity Check Option
  1. Please put the signature file (GnuPG-Helper.bat.asc) in the same place as the .bat file.
  2. Add the GnuPG Key 0xEBE358FC to your keyring.
  3. Press the option "Check this batch file integrity" on the .bat file.
Check for Update Option
  1. Press the option "Check for update" on the .bat file.
  2. Require internet connection
  3. Will download a file to your temp folder to check for version. File will be removed automatically afterwards.
Support for Portable GPG.exe and Portable Keyring
  1. First, do the integrity check
  2. Edit the .bat file using a text editor.
  3. For portable gpg.exe, in line 22, add the full directory to your gpg.exe file after SET PORTABLE=. Example: SET PORTABLE=C:\GPG\gpg.exe
  4. For portable keyring, in line 24, add the full directory to your keyring folder after SET HOMEDIR=. Example: SET HOMEDIR=C:\GPGKeyring\
  5. This will render the signature file GnuPG-Helper.bat.asc invalid.
